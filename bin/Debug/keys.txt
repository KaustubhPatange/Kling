# keycode which is "65" followed by "1" which is enable or disable state.
# Changing this "1" to "0" will exclude this key from showing on the screen.
# The next line is description about the key. This whole configuration takes 2 lines 
# so each key config is separated by an empty line.

# Remember you only need to change "1" value which is whether to display key or not.
# 1 -> Display the key
# 2 -> Don't display the key
